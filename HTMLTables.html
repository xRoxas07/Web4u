HTML Tables

HTML tables permite a los desarrolladores de paginas web a agregar datos en filas y columnas.


Definir una tabla HTML

Una tabla en HTML esta formado por celdas de tabla dentro de filas y columnas

Ejemplo:

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Table cells

Cada tabla en definina por una etiqueta <td> y </td>

Todo entre <td> y </td> es contenido de la celda.

Ejemplo:

             <table>
                <tr>
                     <td>Emil</td>
                     <td>Tobias</td>
                     <td>Linus</td>
                 </tr>
            </table>

Table Headers

Algunas veces las celdas quieren ser encabezados, en ese caso usa la etiqueta <th> en vez de las etiquetas <td>:

Ejemplo:

                    <table>
                        <tr>
                            <th>Person 1</th>
                             <th>Person 2</th>
                             <th>Person 3</th>
                        </tr>
                        <tr>
                            <td>Emil</td>
                             <td>Tobias</td>
                             <td>Linus</td>
                         </tr>
                         <tr>
                            <td>16</td>
                             <td>14</td>
                             <td>10</td>
                         </tr>
                    </table>

Por defecto, el texto en <th> estaran enegricidos y centrados, pero tu puedes cambiar eso con CSS.
