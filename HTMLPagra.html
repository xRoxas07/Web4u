HTML Paragraphs

Un párrafo siempre comienza en una nueva línea y generalmente es un bloque de texto.

HTML Paragraphs

El elemento HTML <p> define un párrafo

Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

Ejemplo:

        <p>This is a paragraph.</p>
        <p>This is another paragraph. </p>

HTML Display

No puede estar seguro de cómo se mostrará el HTML.

Las pantallas grandes o pequeñas y las ventanas redimensionadas generarán resultados diferentes.

Con HTML, no puede cambiar la visualización agregando espacios o líneas adicionales en su código HTML.

El navegador eliminará automáticamente los espacios y líneas adicionales cuando se muestre la página:

Ejemplo:

        <p>
        This paragraph
        contains a lot of lines
        in the source code,
        but the browser
        ignores it.
        </p>

        <p>
        This paragraph
        contains         a lot of spaces
        in the source         code,
        but the        browser
        ignores it.
        </p>

HTML Horizontal Rules

La etiqueta <hr> define una pausa temática en una página HTML y, por lo general, se muestra como una regla horizontal.

El elemento <hr> es usado para separar el contenido (o definir un cambio) en una página HTML:

Example:

        <h1>This is heading 1</h1>
        <p>This is some text.</p>
        <hr>
        <h2>This is heading 2</h2>
        <p>This is some other text.</p>
        <hr>

La etiqueta <hr> es lo que significa que no tiene etiqueta final.

HTML Line Breaks

El elemento HTML <br> define un salto de línea.

Usa <br> si desea un salto de línea (una nueva línea) sin comenzar un nuevo párrafo:

Ejemplo:

        <p>This is<br>with line breaks.</p>

HTML Line Breaks

Este poema se mostrará en un solo:

        <p>
        My Bonnie lies over the ocean.

        My Bonnie lies over the sea.

        My Bonnie lies over the ocean.

        Oh, bring back my Bonnie to me.

        </p>

Solución - El elemento <pre>

El elemento <pre> define texto preformateado.

El texto dentro del <pre> es mostrado en una fuente de ancho fijo (generalmente Courier) y conserva tanto los espacios como los saltos de línea: 

        <pre>
        My Bonnie lies over the ocean.

        My Bonnie lies over the sea.

        My Bonnie lies over the ocean.

        Oh, bring back my Bonnie to me.
        </pre>
