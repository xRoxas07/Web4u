CSS Background

Las propiedades de CSS background se utilizan para agregar efectos de fondo a los elementos.

Nombres de colores

CSS background-color

Las propriedades background-color especifica el color de fondo de un elemento.

Ejemplo:

        body {
            background-color: ligthblue;
        }

Con CSS, un color suele especificarse mediante:

  • un nombre valido - como "red"
  • un valor HEX valido - como "#ff0000"
  • un valor RGB valido - como "rgb(255,0,0)"

Mira CSS Color Values para obtener una lista completa de los posibles valores de color.

Otros elementos

Puede establecer el color de fondo de cualquier elemento HTML:

Ejemplo:

        h1 {
            background-color: green;
        }

        div {
            background-color: ligthblue;
        }

        p {
            background-color: yellow;
        }

Opacidad / Transparencia

La propiedad opacity especifica la opacidad / transparencia de un elemento. Puede tomar un valor de 0,0 a 1,0. Cuanto menor sea el valor, más transparente:

Opacity 1

Opacity 0.6

Opacity 0.3

Opacity 0.1

Ejemplo:

        div {
            background-color: ligthblue;
            opacity: 0.3;
        }

Transparencia usando RGBA

Si no desea aplicar opacidad a los elementos secundarios, como en nuestro ejemplo anterior, use valores de color RGBA. El siguiente ejemplo establece la opacidad del color de fondo y no del texto:

100% opacity

60% opacity

30% opacity

10% opacity

Tu aprendiste del capitulo CSS Colors, que puede utilizar RGB como valor de color. Además de RGB, puede utilizar un valor de color RGB con un canal alfa (RGBA), que especifica la opacidad de un color.

An RGBA color value is specified with: rgba(red, green, blue, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

Ejemplo:

        div {
            background: rgba(0, 128, 0, 0.3) /* Green background with 30% opacity */
        }

CSS background-image

La propiedad background-image especifica una imagen para usar como fondo de un elemento.

De forma predeterminada, la imagen se repite para que cubra todo el elemento.

Ejemplo

Establece la imagen de fondo para una página: 

        body {
            background-image: url("paper.gif");
        }

Ejemplo

Este ejemplo muestra una mala combinación de texto e imagen de fondo. El texto es apenas legible:

        body {
            background-image: url("bgdesert.jpg");
        }
