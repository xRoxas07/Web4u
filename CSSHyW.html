CSS Height and Width

Las propiedades del CSS height y width se utilizan para establecer la altura y el ancho de un elemento.

La propiedad del max-width utiliza para establecer el ancho máximo de un elemento.

CSS Setting height and width

Las propiedades de height y width se utilizan para establecer la altura y el ancho de un elemento.

Las propiedades de alto y ancho no incluyen relleno, bordes ni márgenes. Establece la altura / ancho del área dentro del relleno, el borde y el margen del elemento.

Valores de CSS height and width

Las prpiedades height y width pueden tener los siguientes valores:

  • auto - Esto es predeterminado. El navegador calcula la altura y el ancho.
  • length - Define la altura / ancho en px, cm, etc.
  • % - Define la altura / ancho en porcentaje del bloque contenedor.
  • initial - Establece la altura / ancho a su valor predeterminado.
  • inherit - El alto / ancho se heredará de su valor principal.

Ajustando max-width

La propiedad max-width se utiliza para establecer el ancho máximo de un elemento.

El max-width puede especificar en valores de longitud, como px, cm, etc., o en porcentaje (%) del bloque contenedor, o establecer en ninguno (esto es predeterminado. Significa que no hay un ancho máximo).

El problema con <div> ocurre cuando la ventana del navegador es más pequeña que el ancho del elemento (500px). Luego, el navegador agrega una barra de desplazamiento horizontal a la página.

El max-width en lugar, en estas situaciones, mejorara el manejo del navegador de ventanas pequeñas.

Consejo: Arrastre la ventana del navegador a menos de 500px de ancho para ver la diferencia entre los dos divs.

Este elemento tiene una altura de 100 píxeles y un ancho máximo de 500 píxeles.

Nota: Si por alguna razón usa la propiedad width y la propiedad max-width en el mismo elemento, y el valor de la propiedad width es mayor que la propiedad max-width; se utilizará la propiedad de ancho máximo (y se ignorará la propiedad de ancho).

Ejemplo:

        div {
            max-with: 500px;
            heigth: 100px;
            background-color: powerblue;
        }
