CSS Padding

Padding se utiliza para crear espacio alrededor del contenido de un elemento, dentro de los bordes definidos.

Las propiedades padding se utilizan para generar espacio alrededor del contenido de un elemento, dentro de los bordes definidos.

Con CSS, tienes control total sobre el relleno. Existen propiedades para configurar el relleno para cada lado de un elemento (superior, derecha, inferior e izquierda).

Padding - Lados indiviaduales

CSS tiene propiedades para especificar el relleno para cada lado de un elemento:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Todas las propiedades de relleno pueden tener los siguientes valores:

  • length - specifies a margin in px, pt, cm, etc.
  • % -specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element

Nota: Los valores negativos están permitidos.

        div {
            padding-top: 100px;
            padding-bottom: 100px;
            padding-right: 150px;
            padding-left: 80px;
        }

Padding - Propiedad de taquigrafía

Para acortar el código, es posible especificar todas las propiedades de los márgenes en una propiedad.

La propiedad padding es una propiedad abreviada para las siguientes propiedades de relleno individuales:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Pues así es como funciona:

Si la propiedad padding tiene cuatro valores:

  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px
    • left padding is 100px

Ejemplo:

        div {
            padding:  25px 50px 75px 100px;
        }

Si la propiedad padding tiene tres valores:

  • padding: 25px 50px 75px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px

Ejemplo:

        div {
            padding:  25px 50px 75px;
        }

Si la propiedad padding tiene dos valores:

  • padding: 25px 50px;
    • top padding is 25px
    • right padding is 50px

Example:

        div {
            padding:  25px 50px;
        }

Si la propiedad padding tiene un valor:

  • padding: 25px;
    • top padding is 25px

Ejemplo:

        div {
            padding:  25px;
        }

Padding y el Elemento Width

La propiedad CSS width especifica el ancho del área de contenido del elemento. El área de contenido es la parte que se encuentra dentro del relleno, el borde y el margen de un elemento (the box model).

Entonces, si un elemento tiene un ancho específico, el relleno agregado a ese elemento se agregará al ancho total del elemento. A menudo, este es un resultado no deseado.

Ejemplo:

        div {
            width:  300px;
            padding: 25px;
        }

Para mantener el ancho en 300 px, sin importar la cantidad de relleno, puede usar la propiedad de tamaño de caja. Esto hace que el elemento mantenga su ancho real;
si aumenta el relleno, el espacio de contenido disponible disminuirá.

Ejemplo:

            div {
                width:  300px;
                padding: 25px;
                box-sizing: border-box;
            }
